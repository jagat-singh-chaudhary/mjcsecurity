## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/jagat-singh-chaudhary/MJCSECURITY.git
cd MJCSECURITY
```

### 2. Add Your Photo
- Replace `founder-photo.jpg` with your professional photo
- Make sure the file is in the root directory
- Update the filename in `index.html` if needed

### 3. Customize Content
- Update email: Change `singhjc083@gmail.com` throughout the site
- Update LinkedIn: Change LinkedIn URL
- Modify service descriptions as needed
- Update case studies with real client information
- Add blog posts

### 4. Deploy to GitHub Pages

#### Option A: Automatic (if already set up)
```bash
git add .
git commit -m "Update website content"
git push origin main
```

#### Option B: Manual Setup
1. Go to your GitHub repository
2. Navigate to Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Your site will be live at: `https://jagat-singh-chaudhary.github.io/MJCSECURITY/`

## 📝 Customization Guide

### Change Contact Information
Edit the Contact section in `index.html`:
```html
<a href="mailto:your-email@example.com">Your Email</a>
<a href="https://your-linkedin-url" target="_blank">Your LinkedIn</a>
```

### Update Service Descriptions
Find the Services section and modify the text:
```html
<div class="service-card">
    <i class="fas fa-shield-alt"></i>
    <h3>Your Service Name</h3>
    <p>Your service description here</p>
</div>
```

### Add New Blog Posts
Add new blog cards in the Blog section:
```html
<article class="blog-card">
    <h3>Your Blog Title</h3>
    <p class="date">Your Date</p>
    <p>Your blog excerpt</p>
    <a href="#" class="read-more">Read More →</a>
</article>
```

### Modify Pricing
Update pricing plans in the Pricing section:
```html
<div class="price-card">
    <h3>Your Plan Name</h3>
    <div class="price">$Your Price<span>/month</span></div>
    <!-- Add features -->
</div>
```
